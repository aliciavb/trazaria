# Trazaria - tu log de alimentación diario

Una app **100% privada y open source** para registrar tu alimentación. Sin cuentas, sin nube, sin tracking.

** [Prueba la app aquí](https://trazaria.netlify.app)**

---

##  Instalar en tu móvil

Trazaria es una PWA que funciona offline y guarda tus datos localmente.

**iOS (Safari):** Abre la app  Compartir   "Añadir a pantalla de inicio"

**Android (Chrome):** Abre la app  Menú   "Instalar aplicación"

---

##  Características

-  Registro de comidas con cálculo de calorías y macros
-  Base de datos BEDCA (~50 alimentos españoles)
-  Crea tus propios alimentos personalizados
-  Seguimiento diario de tu progreso
-  Export/Import de datos en JSON
-  **100% privado** - todos tus datos en tu dispositivo
-  Funciona offline

---

##  Privacidad de los datos

 Datos locales (IndexedDB) - sin servidor backend  
 Sin cookies, sin tracking, sin analytics  
 Código abierto y auditable  
 No hay cuentas ni login  
 Tus datos nunca salen de tu dispositivo


**Importante:** Puedes hacer backups desde Database  Export JSON

##  Para devs

```bash
git clone https://github.com/aliciavb/trazaria.git
cd trazaria
npm install
npm run dev
```

**Stack:** React + TypeScript + Vite + Tailwind + Dexie.js (IndexedDB) + PWA
